This repository contains my work for CS107e, a variant of Stanford's CS107 that teaches computer systems concepts through bare metal programming on the Mango Pi MQ-Pro (RISC-V architecture). No operating system. Minimal libraries. Everything from the ground up. Topics covered include the RISC-V architecture, assembly language and machine-level code, the C programming language, compilation and linking, memory management, and peripheral control (GPIO, graphics, sound, keyboards). Assignments build incrementally on a core library, culminating in a functional shell running on hardware.

The `mycode` folder contains all code from the assignments and labs, ultimately creating the `libmymango` library which acts as a functioning OS for a Mango Pi.

The `project` folder contains all code for a Mechanical Rubik's Cube solver which uses stepper motors and a graphics UI to fully solve a scrambled cube.
