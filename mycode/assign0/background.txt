3. An absolute path contains the entire path from the home of the file system. A relative path
   is a path that starts from the current location. An absolute path will always work 
   regardless of your location, while a relative path will only work if you are in the correct
   location. The shorthand ~ means the home directory. The shorthand . refers to the current
   folder. The shorthand .. refers to the parent folder.

7. cd: The cd command navigates to a different directory, following an absolute or a relative
       path. One useful option is that if you enter cd with no arguments, it navigates to 
       the home directory. Example: cd ~/cs107e_home/mycode/assign0/

   pwd: The pwd command outputs the absolute path to the current directory. 
        Example: pwd

   ls: The ls command lists the contents of a directory. When there are no arguments, 
       it prints all of the contents of the current directory. You can also specify a 
       path. Some flags you can add are -l which also prints out permissions, file
       size, last modified date, and filename. Another flag is -a, which lists all 
       files, even if they are hidden. Example: ls -al
 
   cat: The cat command prints the contents of a file to the terminal. If you stack the 
        filenames after each other, you can print the content of multiple files. 
        Example: cat background.txt
 
   mkdir: The mkdir command greates a new folder in the current path with a specified 
          name. Example: mkdir NewFolder

   rmdir: The rmdir command will delete an empty folder/directory. Example: rmdir NewFolder

   touch: The touch command creates a new, empty file. If the file already exists, it will
 	  update the timestamp for the file. You can also create multiple files at the same
	  time. Example: touch newFile1.txt
 
   rm: The rm command permanantly deletes files. You can also delete a directory that is 
       not empty using the -r flag. If there are manual prompts to delete that you want to 
       bypass, you can also use the -f flag. Example: rm -r NewFolder
  
   echo: The echo command displays a string to the terminal screen. This could be a string
	 in quotes or a variable. If you want to display a variable, you must put a $ before
	 the name. If you want to write the string to a file instead, you can use > (overwrite)
	 or >> (append). Example: echo $CS107E
